# Framework Selection

## 目标与框架

| 目标 | 框架 | 参考来源 |
| --- | --- | --- |
| 说服管理层 | SCQA / Pyramid Principle | Barbara Minto, McKinsey |
| 学术论文 | IMRaD | 学术写作通用规范 |
| 技术文档 | 倒金字塔 | 新闻写作 / Divio |
| 商业提案 | Problem-Solution-Impact | 咨询与商业写作 |
| 案例研究 | STAR | 行为案例表达 |

## 写作质量护栏

- **MECE**：论点独立且尽量穷尽
- **So What**：每节都回答“这意味着什么”
- **Data-backed**：关键判断有数据、引用或案例支撑
- **Reader-aware**：根据读者背景调整术语和深度

## 何时改框架

- 写到一半发现章节像材料堆积而不是论证
- 用户新增目标或受众变化
- 数据无法支撑当前结构
